# xhackTool 
![xhackTool preview](xhackTool.jpg)

## introduction
xhackTool is a tool that helps you to install and use multiple penetration testing and hacking tools on Android system with ease.

## Instalations
```
$ pkg update -y && pkg upgrade -y
$ apt update -y && apt upgrade -y
$ pkg install git
$ pkg install python -y
$ git clone https://github.com/Whomrx666/xhackTool.git
$ cd xhackTool 
$ python xhackTool.py
```

## Instructions
- **first**: Install tools according to the instructions above
- **second**: Go to the tool page that you want to select
- **third**: Select one of the tools you use
- **last**: The selected tools will automatically download and be ready to use
